# auto 与 decltype

auto 不只是"偷懒不写类型"——它有完整的推导规则、代理类型陷阱和与模板推导的一致性。decltype 则是模板元编程的基石，让你精确获取表达式的类型。C++17 的 CTAD 更是让模板参数推导变得前所未有地简洁。这一章我们把类型推导的三个维度（auto、decltype、CTAD）彻底搞清楚。
