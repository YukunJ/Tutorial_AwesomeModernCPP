# 属性系统

C++ 的属性（attribute）让你在不改变代码逻辑的前提下，给编译器传递额外的信息——"这个返回值别忽略"、"这个变量可能没用但别报警"、"这个分支更可能执行"……这些信息能帮你捕获 bug、消除无用警告、甚至优化性能。这一章我们覆盖 C++11-17 的标准属性和 C++20-23 的性能导向新属性。
