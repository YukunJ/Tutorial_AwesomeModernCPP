# Lambda 与函数式编程

Lambda 表达式是 C++11 引入的最受欢迎的特性之一——它让匿名函数成为一等公民，彻底改变了我们使用 STL 算法和回调的方式。这一章我们从 lambda 基础语法出发，深入捕获机制的语义与陷阱，探索泛型 lambda 的模板能力，理解 std::function 和 std::invoke 的底层机制，最后在函数式编程模式中体验高阶函数、组合和管道的优雅。
