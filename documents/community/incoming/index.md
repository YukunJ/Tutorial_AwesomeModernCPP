# 社区来稿初刊

这里展示已经通过基础检查、可以公开阅读和讨论的社区来稿。

初刊文章会被线上文档站收录，也可以在 TAMCPP 周报中被引用。它们不代表主线教程的最终定稿，后续可能根据读者反馈、外部平台讨论和维护者审阅继续修改。

## 状态说明

进入本区的文章通常满足：

- 能正常渲染。
- 没有明显技术硬伤。
- 来源、引用、图片和代码说明清楚。
- 作者同意公开展示，并允许维护者进行必要编辑。

## 投稿方式

如果你想投稿，可以提交一个 PR，把 Markdown 文件放到本目录下。文件名建议使用英文小写和连字符，例如：

```text
documents/community/incoming/my-first-modern-cpp-note.md
```

你可以先只提交正文，不必一开始补齐完整 frontmatter、导航和索引。维护者会协助整理。

建议在文章开头写明：

- 标题。
- 作者或署名方式。
- 目标读者。
- 是否原创。
- 主要参考资料。
