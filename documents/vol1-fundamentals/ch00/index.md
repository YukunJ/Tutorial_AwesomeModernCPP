# 环境搭建与第一个程序

万事开头难——但搭环境这件事本身不应该成为拦路虎。这一章我们会把开发环境从零搭起来，无论你用的是 Linux 还是 Windows，都只需要几步就能就绪。然后我们直接上手写第一个 C++ 程序，亲眼看到编译、运行的全过程，对这门语言建立一个最直觉的印象。

本章适合完全没有 C++ 经验的读者。如果你已经有趁手的工具链，可以跳过环境搭建，直接从第一个程序开始。
