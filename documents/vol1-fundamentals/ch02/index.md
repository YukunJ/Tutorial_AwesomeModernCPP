# 控制流

程序的本质是"根据条件做不同的事"和"重复做一些事"。这一章我们把 C++ 里控制程序执行流程的语法全部过一遍——从最基础的 `if`/`else` 到各种循环，再到 C++11 引入的 range-for。你会发现 range-for 写起来比传统的 `for(int i=0; ...)` 舒服太多，后面我们会大量用到它。
