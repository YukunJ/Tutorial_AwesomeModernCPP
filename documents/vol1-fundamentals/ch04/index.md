# 指针与引用

说到 C++ 让人头疼的东西，指针绝对排得上号——但说实话，只要搞明白"地址"和"值"的区别，指针并没有那么可怕。这一章我们会把指针从头讲透，包括指针运算和它与数组的亲密关系；然后引入引用这个更安全的"别名"机制。最后我们提前预览一下智能指针的概念，给你一个"现代 C++ 不需要裸指针"的心理预期。
