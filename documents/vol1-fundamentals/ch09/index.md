# 模板初步

模板是 C++ 实现泛型编程的核心机制——写一份代码，适配多种类型。这一章我们先从函数模板入手，看看编译器怎么帮你自动推导类型参数；然后过渡到类模板，理解它和普通类的区别；最后了解一下模板特化，知道当你需要对某个类型做特殊处理时该怎么做。这里是模板的入门，后面高级卷会深入展开。
