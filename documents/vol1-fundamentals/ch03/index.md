# 函数

函数是组织代码的基本单元，也是我们从"写脚本"过渡到"写工程"的第一步。这一章我们从函数的定义和调用说起，重点搞清楚参数传递的几种方式——按值、按引用、按指针，它们之间的区别是写出高效代码的前提。然后我们看一下函数重载和默认参数怎么配合使用，最后认识 `inline` 和 `constexpr` 函数，这两个关键字在嵌入式和性能敏感场景里出场率极高。
