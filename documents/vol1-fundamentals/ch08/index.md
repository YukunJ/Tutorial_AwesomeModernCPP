# 继承与多态

面向对象的威力在于"用统一的接口操作不同类型的对象"，而实现这一点的核心机制就是继承和多态。这一章我们从单继承讲起，搞清楚虚函数表到底是什么、纯虚函数和抽象类怎么设计接口，然后看看多继承和虚继承的复杂度。最后通过一个实战案例把 OOP 的设计思路串起来。
