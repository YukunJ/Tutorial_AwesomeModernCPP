# 运算符重载

运算符重载让自定义类型像内置类型一样自然地参与运算——你可以让两个 `Vec3` 直接相加、让自定义的矩阵类型支持 `[]` 索引、甚至让对象像函数一样被调用。这一章我们把常用的运算符重载全部过一遍，搞清楚哪些能重载、哪些不能、以及怎么做才不会搞出一堆隐式转换的坑。
