# 类与面向对象

类是 C++ 把"数据"和"操作"打包在一起的机制，也是面向对象编程的基石。这一章我们从最简单的类定义开始，逐步搞清楚构造函数怎么写、析构函数什么时候被调用、`static` 成员和普通成员的区别，以及友元这个"特权通道"是怎么回事。理解了这些，你就能写出自己的封装良好的类型了。
