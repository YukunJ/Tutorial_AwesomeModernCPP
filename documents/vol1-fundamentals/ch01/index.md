# 类型与值类别

类型系统是 C++ 最核心的设计之一——它决定了数据在内存中怎么躺、编译器帮你检查什么、以及你能对一段数据做什么操作。这一章我们从基本数据类型出发，搞清楚 C++ 里的整数、浮点、字符和布尔到底是怎么回事；然后聊一聊类型转换那些容易踩坑的隐式规则，再认识一下 `const` 这个老朋友。最后我们会初步接触"值类别"的概念，为后面理解移动语义打下基础。
